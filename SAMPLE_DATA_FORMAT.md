## 使用說明

### 如何更新資料?

1. **公司清單** (每月更新一次即可)
   - 從證交所/櫃買中心官網下載最新上市櫃公司清單
   - 整理成 CSV 格式
   - 上傳到 GitHub Repository 的 `StockInfo` 資料夾

2. **買超排行** (建議每天更新)
   - 從證交所「三大法人買賣超統計」取得數據
   - 整理成 TXT 格式
   - 上傳到 GitHub Repository 的 `StockInfo` 資料夾

### 快速更新方式

#### 方法 1: 網頁介面更新
1. 前往 GitHub Repository
2. 進入 `StockInfo` 資料夾
3. 點擊要更新的檔案
4. 點擊右上角 ✏️ (編輯) 按鈕
5. 修改內容
6. 點擊 "Commit changes"

#### 方法 2: Git 指令更新
```bash
# 下載 Repository
git pull

# 修改檔案
vim StockInfo/TSE_buy_ranking.txt

# 提交更新
git add StockInfo/
git commit -m "Update buy ranking data"
git push
```
