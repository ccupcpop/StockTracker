# StockInfo 資料夾

此資料夾用於存放所有股票分析相關的資料檔案。

## 必要檔案

### 輸入檔案 (需要手動準備)

1. **tse_company_list.csv** - 上市公司清單
   ```csv
   代碼,名稱,產業
   1101,台泥,水泥工業
   2330,台積電,半導體業
   ```

2. **otc_company_list.csv** - 上櫃公司清單
   ```csv
   代碼,名稱,產業
   5274,信驊,半導體業
   6510,精測,半導體業
   ```

3. **TSE_buy_ranking.txt** - 上市三大法人買超排行
   ```
   #,代碼,名稱,買超張數
   1,2330,台積電,12345
   2,2454,聯發科,6789
   ```

4. **OTC_buy_ranking.txt** - 上櫃三大法人買超排行
   ```
   #,代碼,名稱,買超張數
   1,5274,信驊,5678
   2,6510,精測,4321
   ```

### 輸出檔案 (程式自動生成)

1. **TSE_hotstock_data.json** - 上市熱門股票資料
2. **OTC_hotstock_data.json** - 上櫃熱門股票資料
3. **twstock_news.json** - 台股新聞資料

## 資料來源建議

- **公司清單**: 可從證交所/櫃買中心官網下載
- **買超排行**: 可從證交所三大法人買賣超統計取得
- **新聞資料**: 程式自動從 cmnews.com.tw 爬取

## 注意事項

1. CSV 檔案請使用 UTF-8 編碼
2. 買超排行檔案中的買超張數可以是正數或負數
3. 所有檔案都應該定期更新以確保資料準確性
